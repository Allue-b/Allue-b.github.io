# 🌐 Ruta de Aprendizaje de Matemáticas

Proyecto web que presenta una **ruta estructurada de aprendizaje de matemáticas**, organizada en 10 temas progresivos, desde conceptos básicos hasta fracciones.

El objetivo es ofrecer una experiencia clara, visual y accesible para aprender paso a paso.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Diseño responsive (Flexbox y Grid)

---

## ✨ Funcionalidades

- 📚 Ruta de aprendizaje con 10 temas organizados
- 🌍 Soporte multidioma (español / inglés)
- 🌙 Modo oscuro
- 📱 Diseño responsive (móvil y escritorio)
- 🔗 Navegación interna optimizada
- ❓ Sección de preguntas frecuentes (FAQ)
- 🧭 Generación dinámica de contenido (timeline con JavaScript)
- 🔍 Optimización SEO (meta etiquetas y datos estructurados)
